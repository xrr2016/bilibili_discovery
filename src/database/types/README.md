# `src/database/types`

本目录定义数据库层的标准数据结构，是项目的数据类型真相来源。

主要职责：
- 定义会被数据库持久化或跨模块共享的实体结构
- 为接口层与实现层提供统一类型基础
- 承载项目级共享配置、兼容访问结构等标准类型

设计边界：
- 这里定义“标准数据结构”
- UI 页面自己的临时状态类型不应该放在这里

主要内容：
- `base.ts`：基础公共类型
- `creator.ts` / `video.ts` / `behavior.ts` / `semantic.ts`：核心实体
- `analytics.ts`：分析层实体
- `app.ts`：项目当前使用的应用级共享类型
