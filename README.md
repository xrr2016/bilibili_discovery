# Bilibili Discovery Engine

一个浏览器扩展，帮助您探索B站内容，通过随机推荐UP主、标签分类等方式发现新的有趣内容。

## 主要功能

- 🎲 随机推荐UP主：从您的关注列表中随机推荐UP主
- 🏷️ 智能分类：自动为UP主添加分类标签
- 📊 统计信息：查看关注UP数量、分类词条数、视频数量等统计
- 🔍 筛选功能：根据标签筛选UP主
- 🎨 美观的UI：现代化的界面设计

## 安装方法

### Chrome/Edge浏览器

1. 下载最新发布的 `.crx` 文件
2. 打开浏览器，访问 `chrome://extensions/` 或 `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 将 `.crx` 文件拖拽到扩展页面
5. 确认安装

### Firefox浏览器

1. 下载最新发布的 `.xpi` 文件
2. 打开浏览器，访问 `about:addons`
3. 点击右上角的齿轮图标，选择"安装附加组件从文件"
4. 选择下载的 `.xpi` 文件
5. 确认安装

### 开发者安装

从源码安装：

```bash
# 克隆仓库
git clone https://github.com/yourusername/bili-random-up.git
cd bili-random-up

# 安装依赖
npm install

# 构建项目
npm run build

# 在浏览器中加载 dist/extension 目录
```

## 使用方法

### 初始设置

1. 首次使用时，需要设置您的B站用户ID
   - 点击浏览器工具栏中的插件图标
   - 在弹出的窗口中输入您的B站用户ID
   - 点击"保存设置"

### 更新关注列表

1. 点击插件图标打开弹窗
2. 点击"更新关注列表"按钮
3. 等待更新完成

### 查看统计信息

1. 点击插件图标
2. 点击"查看统计"按钮
3. 在统计页面中查看：
   - 关注UP数量
   - UP分类词条数
   - 视频数量
   - 关注UP列表（点击头像或名称可跳转到UP主页）
   - UP分类词条（支持搜索和拖拽筛选）

### 随机推荐

1. 点击插件图标
2. 点击"随机推荐"按钮
3. 浏览器会自动打开推荐UP的视频

## 开发

### 项目结构

```
bili-random-up/
├── extension/          # 扩展源码
│   ├── api/          # B站API接口
│   ├── background/    # 后台服务
│   ├── content/      # 内容脚本
│   ├── engine/       # 分类和推荐引擎
│   ├── storage/      # 存储管理
│   └── ui/          # 用户界面
├── dist/            # 构建输出
├── scripts/         # 构建脚本
└── package.json
```

### 可用命令

```bash
# 安装依赖
npm install

# 构建项目
npm run build

# 运行测试
npm run test

# 探测B站API
npm run probe:bili
```

## 权限说明

- `storage`：用于存储用户设置和UP列表数据
- `tabs`：用于打开新标签页显示推荐内容
- `alarms`：用于定时更新数据

## 隐私说明

本扩展仅访问B站公开数据，不会收集或上传任何个人信息。所有数据均存储在本地浏览器中。
